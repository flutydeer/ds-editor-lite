# 资源文件修改

## 翻译文件

在项目根目录配置 `debug` preset 后，从源代码获取待翻译的字符串并保存：

```sh
cmake --build --preset debug --target DsEditorLite_lupdate
```

在 Qt 语言家编辑并保存后，发布成二进制文件：

```sh
cmake --build --preset debug --target DsEditorLite_lrelease
```

`translation_zh_CN.qm` 由 CMake 在构建目录中生成并通过 `/i18n` 资源前缀嵌入应用，
无需手工生成或提交到仓库。Windows 部署只携带 `zh_CN` 的 Qt 标准控件翻译。

应用运行时通过安装或卸载 translator 切换 UI 语言。常驻 Widget 应处理标准
`QEvent::LanguageChange` 并在 `retranslateUi()` 中更新长期持有的 Label、Action、Menu、
Tooltip 和组合框显示文本；动态菜单和对话框继续在创建时调用 `tr()`。
